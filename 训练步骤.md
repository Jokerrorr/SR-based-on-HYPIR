# v4 两阶段训练步骤

## Stage 1: 对齐预训练（~6000步）

```bash
conda activate hypir
cd SR-based-on-HYPIR
accelerate launch train.py --config configs/sd2_alignment_stage1_train.yaml
```

- **冻结**：RM、VAE、UNet+LoRA（HYPIR 预训练）
- **训练**：AlignmentHandler
- **损失**：L1(noise_pred, noise)
- **输出**：`output_alignment_stage1/checkpoint-6000/state_dict.pth`（仅 alignment 权重）

## Stage 2: 对抗微调（~5000步）

先修改配置，将 `alignment_pretrained_path` 指向 Stage 1 输出：

```yaml
# configs/sd2_alignment_train.yaml
alignment_pretrained_path: output_alignment_stage1/checkpoint-6000/state_dict.pth
```

然后运行：

```bash
accelerate launch train.py --config configs/sd2_alignment_train.yaml
```

- **冻结**：RM、VAE
- **训练**：UNet LoRA（从 HYPIR 预训练微调）+ AlignmentHandler（从 Stage 1 微调）
- **损失**：λ_l2·MSE + λ_lpips·LPIPS + λ_gan·D + λ_align·L1(noise_pred, noise)
- **训练方式**：G/D 交替训练
- **输出**：`output_alignment/checkpoint-*/state_dict.pth`（LoRA + alignment 权重）

## 观察训练

```bash
# 损失曲线
tensorboard --logdir output_alignment_stage1/logs   # Stage 1
tensorboard --logdir output_alignment/logs           # Stage 2

# CSV 日志
cat output_alignment_stage1/loss_log.csv   # step, noise_loss
cat output_alignment/loss_log.csv          # step, G_total, G_mse, G_lpips, G_disc, G_align, D
```
