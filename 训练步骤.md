● 两阶段训练步骤

  Stage 1: 对齐预训练（~3000步）

  conda activate hypir
  cd SR-based-on-HYPIR
  accelerate launch train.py --config configs/sd2_alignment_stage1_train.yaml

  冻结：RM、VAE、UNet+LoRA(HYPIR预训练) → 只训练Alignment Handler

  输出：output_alignment_stage1/checkpoint-3000/state_dict.pth

  Stage 2: 联合微调（~5000步）

  先修改配置，将alignment_pretrained_path指向Stage 1输出：

  # configs/sd2_alignment_train.yaml 第83行
  alignment_pretrained_path:
  output_alignment_stage1/checkpoint-3000/state_dict.pth

  然后运行：

  accelerate launch train.py --config configs/sd2_alignment_train.yaml

  冻结：RM、VAE → 可训练：UNet LoRA(从HYPIR预训练微调) +
  Alignment(从Stage1微调)

  观察训练

  # 损失曲线
  tensorboard --logdir output_alignment_stage1/logs   # Stage 1
  tensorboard --logdir output_alignment/logs           # Stage 2

  # CSV日志
  cat output_alignment_stage1/loss_log.csv
  cat output_alignment/loss_log.csv
