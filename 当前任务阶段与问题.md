## 当前状态：v3 改造完成 ✅

### v3 核心变更：FaithDiff训练 + 单步推理

v2 版本使用了 20 步 DDPM 多步去噪推理，严重拖慢速度。
v3 保持 HYPIR 单步推理速度，同时使用 FaithDiff 式噪声预测训练。

**设计决策**：
1. 训练时固定 t=200（与 HYPIR 原始设计一致）
2. 推理时单步：z_lq 加噪到 t=200 → UNet 一步预测 → scheduler 一步得到结果
3. Stage 1 可训练模块与 FaithDiff 一致（alignment handler 全部参数）

### 数据流

**训练**：
```
GT → VAE.encode → z_hq → add_noise(t=200) → noisy_hq ──→ UNet ──→ noise_pred
                                                        ↑              ↑
LQ → RM(冻结) → x_rm → VAE.encode → x_en → AlignmentHandler → features
                                                        ↓
                                              Loss = L1(noise_pred, noise)
```

**推理（单步）**：
```
z_lq = VAE.encode(LQ)
x_en = VAE.encode(RM(LQ))
z_in = add_noise(z_lq, noise, t=200)     # 匹配训练噪声水平
noise_pred = UNet(z_in, t=200, text, alignment_features)
z_pred = scheduler.step(noise_pred, 200, z_in).pred_original_sample
HQ = VAE.decode(z_pred)
```

### 与原实现/FaithDiff 对比

| 要素 | 原错误实现 | FaithDiff | 我们 v3 |
|------|-----------|-----------|---------|
| UNet 训练输入 | z_lq | noisy_hq (随机t) | noisy_hq (固定t=200) |
| 时间步 | 固定 t=200 | 随机 randint(0,T) | 固定 t=200 |
| Loss | GAN+L2+LPIPS | L1(noise_pred, noise) | L1(noise_pred, noise) |
| 推理 | 单步 | 多步 DDPM | 单步 |
| 速度 | 快 | 慢(~20x) | 快(与HYPIR相同) |
